According to PANews, Arkham has marked the address starting with 1JbezD as the
cold wallet address of Mt. Gox. All
[BTC](/en/trade/BTC_USDT?contentId=8707566753633) from the previous cold
wallet address of Mt. Gox has been consolidated to this address today. The Mt.
Gox: 1JbezD address then carried out further distribution. 4000 BTC were sent
to the address starting with 152wz8, which was then transferred to the address
starting with 1AsHPP. 21.946 BTC were sent to the address starting with
1LsC3V, which was then transferred to the address starting with 1AsHPP. 71025
BTC were sent to the address starting with 1LsC3V. 66639 BTC were sent to the
address starting with 16eAGJ, totaling 141.68 thousand BTC. As of 18:50
Beijing time, all bitcoins in the Mt. Gox: 1JbezD address have been
transferred out, and the current balance is zero.

Previously, the MtGox wallet address had transferred several billion dollars
worth of bitcoin to the new address starting with 1JbezD. Mt. Gox confirmed
that this was in preparation for repayment before the creditor repayment
deadline on October 31.
