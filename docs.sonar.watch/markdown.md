##

**What is SonarWatch ?**

SonarWatch is a company that aims to facilitate the **adoption of DeFi** on
multiple chains by providing **services for end-user** but also **for
builders**.

SonarWatch is composed of 2 services :

##

1\. Dashboard

It covers every financial assets (tokens, liquidity pools tokens, NFTs, locked
assets, lent and borrowed assets, etc...) across multiple blockchains.

Learn more in this section : [Explore the web3](/dashboard/connect-your-
public-key)

##

2\. API

This API provide detailed data on DeFi holdings of any address and can be used
to power financial apps and Dapps for both web2 and web3 companies.

Learn more in this section : [Introduction](/api/introduction)
