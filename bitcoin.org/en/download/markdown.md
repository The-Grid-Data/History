## Check your bandwidth and space

Bitcoin Core initial synchronization will take time and download a lot of
data. You should make sure that you have enough bandwidth and storage for the
block chain size (7GB). If you have a good Internet connection, you can help
strengthen the network by keeping your PC running with Bitcoin Core and port
8333 open. Read the [full node guide](/en/full-node) for details.

Bitcoin Core is a community-driven [free
software](https://www.fsf.org/about/what-is-free-software) project, released
under the [MIT license](http://opensource.org/licenses/mit-license.php).
