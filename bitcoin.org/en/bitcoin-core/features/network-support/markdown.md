# Donate Bandwidth Using Bitcoin Core

![Support the network](/img/bitcoin-core/slider-network.svg?1716491272)

[Download Bitcoin Core](/en/download)

Bitcoin Core 27.0

![Windows](/img/os/windows.png?1716491272) ![Mac](/img/os/mac.png?1716491272)
![Linux](/img/os/linux.png?1716491272)

* * *

The Bitcoin peer-to-peer network serves both Bitcoin Core and many other
Bitcoin programs (mostly lightweight wallets). By contributing some of your
bandwidthâtypically about 100 GB upload a monthâyou can help support
Bitcoin.

The [bandwidth sharing guide](/en/full-node) provides all of the details you
need to begin donating bandwidth.

## Donât Forget About Decentralization

The Bitcoin network needs more than bandwidthâit also needs people who
[actively secure](/en/bitcoin-core/features/validation#do-you-validate) their
bitcoins using Bitcoin Core. By securing your bitcoins with a full node like
Bitcoin Core, you [help protect Bitcoinâs decentralization](/en/bitcoin-
core/features/validation#help-protect-decentralization) for yourself and other
Bitcoin users.

You can help protect decentralization instead of donating bandwidth by simply
using Bitcoin Core as your main wallet. Or, even better, you can both donate
bandwidth and protect decentralization at the same time by using Bitcoin Core
as your main wallet while also following the instructions in the [bandwidth
sharing guide](/en/full-node).

## Thank You

Whether you choose to donate bandwidth, protect decentralization, or both,
please know that your fellow Bitcoin users thank you. Without volunteers like
you, Bitcoin would never have come as far as it has.
