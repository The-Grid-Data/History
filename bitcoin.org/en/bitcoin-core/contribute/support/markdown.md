# Supporting Bitcoin Core Users

This site tells Bitcoin Core users where they can [go to find
help](/en/bitcoin-core/help)âbut that help is provided by volunteers like
you. Many questions are asked by complete newbies, so nearly everyone with any
experience using Bitcoin Core can provide help.

Before you start providing support, you may want to familiarize yourself with
the available [documentation](https://developer.bitcoin.org).

## Forums

The following forums are where we believe most users ask their Bitcoin Core
questions.

  * [Bitcoin StackExchange](http://bitcoin.stackexchange.com) is a community dedicated entirely to answering questions about Bitcoin and related technology. Many questions about Bitcoin Core can be found under the [Bitcoin-Qt tag](http://bitcoin.stackexchange.com/questions/tagged/bitcoin-qt)

  * [BitcoinTalk Technical Support](https://bitcointalk.org/index.php?board=4.0) is a sub-forum dedicated to providing help for Bitcoin Core and other Bitcoin programs.

  * [/r/Bitcoin](http://www.reddit.com/r/Bitcoin) is a Reddit community that occasionally features questions about Bitcoin Core. Currently, very few questions receive enough upvotes to be featured on the front page, but you can always check the [new queue](http://www.reddit.com/r/Bitcoin/new) or answer questions on the popular Mentor Monday thread (usually posted around noon UTC on Monday).

  * [/r/BitcoinBeginners](http://www.reddit.com/r/bitcoinbeginners) is a Reddit community that prominently features questions from inexperienced Bitcoin users.

## Live

Internet Relay Chat (IRC) is the most popular way to get live online help with
Bitcoin Core. When providing links to documentation, most chatrooms require
you post full links (not links shortened using services like TinyURL or
Bit.ly).

The links below will get you started using an IRC web interface. Many serious
IRC users use a [native IRC
client](https://en.wikipedia.org/wiki/List_of_IRC_clients).

  * [#bitcoin](https://webchat.freenode.net/?channels=bitcoin&uio=d4) is where most users should ask general questions about Bitcoin Core.

  * [#bitcoin-mining](https://webchat.freenode.net/?channels=bitcoin-mining&uio=d4) hosts discussion about Bitcoin mining, including decentralized mining using Bitcoin Core as part of the system.

  * For more channels, please see the [comprehensive listing](https://en.bitcoin.it/wiki/IRC_channels) on the Bitcoin Wiki.
