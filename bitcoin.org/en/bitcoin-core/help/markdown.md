# Getting Help For Bitcoin Core

There are many ways to get help for Bitcoin Core, including documentation,
forums, and live chatrooms.

_To report an issue, please see the[bug reporting](/en/bitcoin-
core/contribute/issues) page._

## Documentation

Bitcoin Core currently doesnât have any cohesive or complete
documentationâbut we hope to improve that situation soon. For now, you can
use the following resources:

  * Bitcoin Wiki pages: [running Bitcoin](https://en.bitcoin.it/wiki/Running_Bitcoin), [data directory](https://en.bitcoin.it/wiki/Data_directory), and other articles in the [Bitcoin Core documentation category](https://en.bitcoin.it/wiki/Category:Bitcoin_Core_documentation).

  * The [developer reference](https://developer.bitcoin.org/reference/rpc/) provides complete documentation of the RPCs that can be used with `bitcoin-cli` or in third-party programs.

  * The [bandwidth sharing guide](/en/full-node) describes installing Bitcoin Core in detail as well as opening port 8333 to allow other Bitcoin programs to download blocks and transactions from you.

## Forums

Bitcoin has a wide range of [communities](/en/community), but the following
places are the best place to ask for help using Bitcoin Core:

  * [Bitcoin StackExchange](http://bitcoin.stackexchange.com) is a community dedicated entirely to answering questions about Bitcoin and related technology. Many questions about Bitcoin Core can be found under the [Bitcoin-Qt tag](http://bitcoin.stackexchange.com/questions/tagged/bitcoin-qt)

  * [BitcoinTalk Technical Support](https://bitcointalk.org/index.php?board=4.0) is a sub-forum dedicated to providing help for Bitcoin Core and other Bitcoin programs.

  * [/r/BitcoinBeginners](http://www.reddit.com/r/bitcoinbeginners) is a Reddit community for users who have questions about anything Bitcoin-related, including Bitcoin Core.

## Live

Internet Relay Chat (IRC) is the most popular way to get live online help with
Bitcoin Core. When you join an IRC chatroom, you must read the topic (which is
usually automatically displayed) to learn the rules for that chatroom.

  * [#bitcoin](https://webchat.freenode.net/?channels=bitcoin&uio=d4) is the best place to ask general questions about Bitcoin Core.

  * [#bitcoin-mining](https://webchat.freenode.net/?channels=bitcoin-mining&uio=d4) hosts discussion about Bitcoin mining, including decentralized mining using Bitcoin Core as part of the system.

  * For more channels, please see the [comprehensive listing](https://en.bitcoin.it/wiki/IRC_channels) on the Bitcoin Wiki.
