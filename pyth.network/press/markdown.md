## About Pyth

The Pyth Network is a specialized oracle solution for latency-sensitive
financial data that is typically kept behind the "walled gardens" of
centralized institutions.

The Pyth network is focused on finding a new and inexpensive way to bring this
unique data on-chain and aggregating it securely. For more information about
the Pyth network, please visit [pyth.network](/)
