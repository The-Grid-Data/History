# The Games Toolkit on Solana

Create a world where players can play and own, enabling new onchain economies.
True digital ownership is changing the dynamics of video games. Players can
play and own by building their own digital assets and features. Solana’s
scalability makes the massive, player-powered digital economies of future
games possible, today.

[Start Building](https://solanacookbook.com/gaming/hello-world.html)

## A suite of tools to enable scalable blockchain games

## GameShift, by Solana Labs

Compiled by Solana Labs, GameShift is a unified blockchain game development
API aimed at alleviating major pain points games publishers experience when
developing or porting games into web3.

[Learn more](https://gameshift.solanalabs.com/)

![](/_next/image?url=https%3A%2F%2Fcdn.builder.io%2Fapi%2Fv1%2Fimage%2Fassets%252Fce0c7323a97a4d91bd0baa7490ec9139%252Fb57903c93f9545fc807b8e4beaed04fc&w=1200&q=75)

## xNFTs, by Coral

Run games and programs directly from a compatible wallet. Executable NFTs, or
xNFTs, from Coral, allows businesses to create game experiences beyond the
traditional web stack.

[Read xNFT docs](https://docs.xnfts.dev/)

![](/_next/image?url=https%3A%2F%2Fcdn.builder.io%2Fapi%2Fv1%2Fimage%2Fassets%252Fce0c7323a97a4d91bd0baa7490ec9139%252F8ce3c7c7567b497a9851f6b57e5629f7&w=1200&q=75)

## SFTs

SFTs, or semi-fungible tokens, are commonly used in blockchain games. They
start out as fungible before becoming non-fungible after being used.

[Read SFT docs](https://docs.metaplex.com/programs/token-
metadata/overview#semi-fungible-tokens)
